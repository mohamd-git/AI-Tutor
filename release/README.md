# Slide Tutor — Android app

`SlideTutor.apk` is the installable Android version of Slide Tutor. It is a small
wrapper that opens the live app (https://ai-tutor-flame.vercel.app) full screen,
so it always shows the latest version and needs an internet connection.

## Install it on your phone

1. On your phone, open the direct download link:
   https://github.com/mohamd-git/AI-Tutor/raw/main/release/SlideTutor.apk
2. Open the downloaded `SlideTutor.apk`.
3. Android will warn about installing from an unknown source — tap **Settings**,
   allow it for your browser (or Files app), then go back and tap **Install**.
4. Open **Slide Tutor** from your home screen.

Installing outside the Play Store like this is normal for a personal app. The app
is signed with your own key (kept private, not in this repository).
